\# Vendor Performance Analysis



\## Project Overview

This project analyzes vendor performance using Python, SQL, and Power BI.  

The goal is to evaluate vendor profitability, sales contribution, and inventory efficiency.



\## Tools \& Technologies

\- Python

\- Pandas

\- SQLite

\- Seaborn

\- Matplotlib

\- Power BI



\## Project Workflow

CSV Data → Python Data Pipeline → SQLite Database → Exploratory Data Analysis → Vendor Performance Insights → Power BI Dashboard



\## Key Metrics

\- Total Sales

\- Total Purchase

\- Gross Profit

\- Profit Margin

\- Stock Turnover

\- Sales to Purchase Ratio



\## Key Insights

\- MARTIGNETTI COMPANIES and CONSTELLATION BRANDS INC are top vendors by sales.

\- Some brands have high profit margins but low sales, indicating promotional opportunities.

\- Vendor purchase contributions are concentrated among a few major suppliers.



\## Files in Repository

\- `Vendor Performance Analysis.ipynb` → Main analysis

\- `Exploratory Data Analysis.ipynb` → Data exploration

\- `ingestion\_db.py` → Data ingestion pipeline

\- `vendor\_sales\_summary.csv` → Processed dataset
